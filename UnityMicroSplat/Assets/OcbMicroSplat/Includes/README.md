# MicroSplat raw unity shader bits

The parts here are derived work from MicroSplat by Jason Booth.

https://assetstore.unity.com/packages/tools/terrain/microsplat-96478

About 90% of the code included here is part of the free core package.
A few bits are subject to paid plugins and I'm only providing these here to
have it under source control and for people to play with the existing compile
time options. If anyone wants to build up on this and redistribute any further
derived work, they must acquire licenses ($50) for the 3 MicroSplat plugins!

- https://assetstore.unity.com/packages/tools/terrain/microsplat-triplanar-uvs-96777
- https://assetstore.unity.com/packages/tools/terrain/microsplat-anti-tiling-module-96480
- https://assetstore.unity.com/packages/tools/terrain/microsplat-runtime-procedural-texturing-143039
